Sobre Evy

Evy es un toolkit front end compuesto de clases CSS y plugins jQuery. Funciona como soporte en el desarrollo de sitios webs.

Tamaños de pantalla

  • xs representa Extra Small(pantallas con un acho menor a 480px)
  • sm representa Small(pantallas con un acho mayor a 480px y menor a 768px)
  • md representa Medium(pantallas con un acho mayor a 768px y menor a 1024px)
  • lg representa Large(pantallas con un acho mayor a 1024px)

Contenedores

.e-container define una elemento centrado y con un ancho máximo de 1200px. Permite tener un límite cuando las pantallas son muy grandes.

.e-content define un elemento con un padding de 10px. Se utiliza como un simple contenedor de datos

Alineación

Texto a la izquierda con .e-left
Texto al centro con .e-center
Texto a la derecha con .e-right

Tamaño de fuente

Existen tamaños de fuentes predefinidos desde 0.5em a 5em. Definidas cada 0.1em hasta 2em y cada 0.5em desde allí.

Cada clase puede ser aplicada a todos los dispositivos (.e-font-#numero) o mediante las clases para diferenciar tamaños de pantalla (.e-font-xs-#numero, .e-font-sm-#numero, .e-font-md-#numero, .e-font-lg-#numero)

Algunos ejemplos aplicables a todas las pantallas(.e-font equivale a .e-font-xs)
.e-font-0-5 .e-font-1 .e-font-1-1 .e-font-1-3 .e-font-2-5 .e-font-3 .e-font-5

Los siguientes textos solo aplicaran el tamaño definido en pantallas con ancho mayor a 480px y menor a 768px
.e-font-0-5 .e-font-1 .e-font-1-1 .e-font-1-3 .e-font-2-5 .e-font-3 .e-font-5

Colores

Evy brinda una paleta de colores básica, basada en la de Semantic-UI

.e-#color define #color como color de texto.

.e-bg-#color define #color como color de fondo.

Colores disponibles

  • Orange
  • Yellow
  • Green
  • Teal
  • Blue
  • Purple

Otros

A continuación se muestran clases de apoyo

.e-pointer coloca cursor:pointer al elemento al que se le asigne. Pasar mouse por encima de este texto

.e-hidden coloca display:none al elemento al que se le asigne para que no sea visible.

.e-min-lg hace que el elemento solo se vea en pantallas con mas de 1023px de ancho.

.e-min-md hace que el elemento solo se vea en pantallas con mas de 767px de ancho.

.e-min-sm hace que el elemento solo se vea en pantallas con mas de 479px de ancho.
